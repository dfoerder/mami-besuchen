# Mami besuchen – Anleitung 👵💚

Mit dieser kleinen App tragen wir uns ein, **wer Mami wann besucht**, damit sie
jeden Tag Besuch bekommt. Es gibt pro Tag vier Zeiten: **Morgen, Mittag,
Nachmittag, Abend**. Jeder von uns hat einen eigenen Knopf; zusätzliche Besucher
(z. B. Enkel oder Bekannte) kann man mit Namen von Hand eintragen.

**Link:** https://dfoerder.github.io/mami-besuchen/

Kein Konto, kein Passwort, keine Anmeldung – einfach Link öffnen und loslegen.
Alle sehen denselben Kalender, Änderungen erscheinen sofort bei allen anderen.

---

## Als App aufs Handy (empfohlen)

So liegt sie wie eine echte App auf dem Startbildschirm – ein Tipp genügt.

### iPhone / iPad (Safari)
1. Link **in Safari** öffnen (nicht Chrome).
2. Unten auf das **Teilen-Symbol** tippen (Quadrat mit Pfeil nach oben ⬆️).
3. Etwas nach unten scrollen → **„Zum Home-Bildschirm"**.
4. Oben rechts auf **„Hinzufügen"**.
→ Das Herz-Symbol „Mami besuchen" liegt jetzt auf dem Startbildschirm.

### Android (Chrome)
1. Link **in Chrome** öffnen.
2. Oben rechts auf das **Drei-Punkte-Menü ⋮** tippen.
3. **„App installieren"** bzw. **„Zum Startbildschirm hinzufügen"** wählen.
4. Bestätigen.
→ Das Herz-Symbol liegt jetzt auf dem Startbildschirm.

---

## So trägst du dich ein

1. Oben im **Streifen den Tag** antippen (oder mit den Pfeilen ‹ › blättern,
   „Heute" springt zum heutigen Tag).
2. Bei der passenden Zeit (Morgen/Mittag/Nachmittag/Abend) auf **deinen Namen**
   tippen – er wird farbig = du kommst. Nochmal tippen = wieder austragen.
3. **Andere Person** (z. B. ein Enkel): ins Feld „Weitere Person …" den Namen
   schreiben und auf **＋** tippen. Mit dem **×** wieder entfernen.

## Die Punkte unter den Tagen
Unter jedem Tag siehst du **vier kleine Punkte** = die vier Zeiten.
- **Grün** = für diese Zeit ist jemand eingetragen.
- **Grau** = noch frei.

So sieht man sofort, wo Mami noch Besuch fehlt. Steht oben ein gelber Hinweis,
ist für diesen Tag noch **gar niemand** eingetragen.
